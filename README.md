# ClipView

图片裁剪与电影风格黑边合成工具，纯浏览器端运行，无需安装。

## 功能

- **裁剪** — 5 种预设比例（2.35:1 / 16:9 / 4:3 / 1:1 / 21:9）+ 自定义宽高比
- **黑边合成** — 可选开关，支持 16:9 和 4:3 两种输出比例，关闭则直接导出裁剪原图
- **字幕叠加** — 兼容组合下在黑边区居中显示自定义文字，字号随分辨率等比缩放
  - 16:9 黑边 + 2.35:1 / 21:9 裁切
  - 4:3 黑边 + 16:9 裁切
- **实时预览** — 右侧同步显示最终输出效果
- **旋转翻转** — 90° 旋转、水平/垂直翻转，一键重置
- **多方式上传** — 点击选择、拖拽文件、Ctrl+V 粘贴截图
- **导出** — JPEG / PNG / WebP 格式可选，质量滑块可调
- **主题切换** — 亮色/暗色模式，偏好自动保存
- **移动端适配** — 响应式布局

## 使用

1. 打开 `index.html`
2. 上传图片（拖入 / 点击 / Ctrl+V 粘贴）
3. 选择裁切比例，拖动裁剪框调整区域
4. 开启「添加黑边」，选择输出比例（16:9 / 4:3）；或关闭黑边直接导出原比例图
5. 若兼容，输入字幕文字，自动呈现在下方黑边区
6. 选择导出格式与质量，点击下载

## 兼容组合速查

| 黑边比例 | 可加字幕的裁切比例 |
|---------|-----------------|
| 16:9 | 2.35:1, 21:9 |
| 4:3 | 16:9 |

## 技术栈

纯静态页面，依赖 [Cropper.js](https://github.com/fengyuanchen/cropperjs) (CDN)，所有处理在浏览器 Canvas 完成。
