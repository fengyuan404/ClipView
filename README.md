# ClipView

图片裁剪与电影风格黑边合成工具，纯浏览器端运行，无需安装。

## 功能

- **裁剪** — 支持 5 种预设比例（2.35:1 / 16:9 / 4:3 / 1:1 / 21:9）及自定义宽高比
- **实时预览** — 裁剪时右侧同步显示 16:9 黑边最终效果
- **旋转翻转** — 90° 旋转、水平/垂直翻转，支持一键重置
- **多方式上传** — 点击选择、拖拽文件、Ctrl+V 粘贴截图
- **导出** — JPEG / PNG / WebP 格式可选，质量可调
- **主题切换** — 亮色/暗色模式，偏好自动保存
- **移动端适配** — 响应式布局，手机可用

## 使用

1. 打开 `index.html`
2. 上传图片（拖入 / 点击 / 粘贴）
3. 调整裁剪框，选择比例
4. 右侧实时预览 16:9 黑边效果
5. 选择格式和质量，点击导出下载

## 技术栈

纯静态页面，依赖 [Cropper.js](https://github.com/fengyuanchen/cropperjs) (CDN)，所有处理在浏览器 Canvas 完成。
