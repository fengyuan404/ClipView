使用说明

  1. 上传图片：拖入 / 点击 / Ctrl+V 粘贴
  2. 选择裁切比例：点击预设按钮或输入自定义宽高比
  3. 拖动裁剪框调整区域，支持旋转与翻转
  4. 开启「添加黑边」，选择输出比例（16:9 / 4:3）；关闭则直接导出裁切原图
  5. 若兼容，在字幕框中输入文字，自动呈现在下方黑边区
  6. 选择导出格式与质量，点击「导出下载」

功能介绍

  • 裁剪：2.35:1 / 16:9 / 4:3 / 1:1 / 2.39:1 预设 + 自定义宽高比
  • 黑边合成：可选开关，16:9 或 4:3 输出
  • 字幕叠加：兼容组合下黑边区居中显示自定义文字
  • 实时预览：右侧同步显示最终输出效果
  • 旋转翻转：90° 旋转、水平/垂直翻转、一键重置
  • 多方式上传：点击、拖拽、Ctrl+V 粘贴
  • 导出：JPEG / PNG / WebP，质量 10–100% 可调
  • 主题切换：亮色/暗色，偏好自动保存
  • 移动端适配：响应式布局

关于本站

ClipView 是一款纯浏览器端图片裁切与黑边合成工具。无需安装，所有处理在本地 Canvas 完成，图片不会上传到任何服务器。

开源地址：github.com/fengyuan404/ClipView
